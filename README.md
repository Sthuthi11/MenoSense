# Website for Early Detection of Menopause using IoT and DL

## Steps to run:
1. Download the ZIP file from GitHub or clone the repo
2. cd MenoSense
3. npm install
4. npm start

## Features:
1. Connect IoT Device:
   - Currently under development
   - Will allow users to connect IoT-based sensors
   - Enables real-time monitoring of menopause-related physiological data
2. Predict Stage:
   - Users can complete a structured questionnaire
   - The system predicts the current menopause stage
   - Powered by an ensemble deep learning model (backend served via Streamlit)
3. Profile:
   - Users can create and manage their profile
